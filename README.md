# Guess the Number Game

A fun, interactive number guessing game with multiple difficulty levels. Test your intuition and try to guess the secret number before running out of attempts!

## Game Description

The computer randomly selects a number between 1 and 100. Your task is to guess the number within a limited number of attempts. After each guess, you'll receive feedback whether your guess was "Too High" or "Too Low". Choose your difficulty level and see if you can beat the game!

## Features

- Three Difficulty Levels:
  - Easy: 10 attempts
  - Medium: 7 attempts  
  - Hard: 5 attempts

- Interactive Gameplay:
  - Real-time feedback on guesses
  - Attempt counter showing remaining tries
  - Input validation for invalid entries
  - Game replay option without restarting

- User-Friendly Interface:
  - Clear visual formatting
  - Step-by-step guidance
  - Game over messages with the revealed number

## Requirements

- Python 3.x installed on your system
- Jupyter Notebook (for .ipynb file) OR any Python IDE
- No additional packages needed (uses only built-in `random` module)
